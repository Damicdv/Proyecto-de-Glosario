HTML

HTML - Lenguaje de marcado utilizado para crear páginas web y estructurar contenido en la web.

  • Tag: Basic HTML element that defines a page element, such as a title, text, image, or link.
  • Element: Basic HTML component that can be an opening tag, a closing tag, or content.
  • Attribute: Additional information that can be added to an HTML tag to provide information about the element.
  • Block Element: HTML element that creates a new line on the page and can contain other elements.
  • Inline Element: HTML element that is inserted within another tag and does not create a new line on the page.
  • Head: Section of an HTML page that contains information about the document, such as the title and metadata.
  • Body: Section of an HTML page that contains the visible content of the page.
  • Hyperlink: HTML element that allows users to navigate between web pages.
  • Image: HTML element that displays an image on the web page.
  • List: HTML element that displays a list of items, such as a navigation menu or a list of products.
  • Form: HTML element that allows collecting user input, such as names, addresses, or passwords.
  • Input: HTML element that allows the user to enter information, such as text, numbers, or select options.
  • Button: HTML element that creates a button on the page, which can be used to submit forms or trigger functions.
  • Semantic HTML: Practice of using HTML elements that provide clear meaning, improving accessibility and indexing.
  • Header: HTML element that contains presentation and navigation information at the top of the page.
  • Footer: HTML element that contains additional information at the bottom of the page.
  • Sidebar: HTML element that contains secondary or related content in a side area of the page.
  • Section: HTML element that defines a section or content area on the page.
  • Article: HTML element that represents an independent article or entry on the page.
  • Nav: HTML element that creates a navigation menu on the page.
  • Div: Generic HTML element that allows grouping and organizing other elements on the page.
  • Span: Generic HTML element that allows applying style or behavior to a set of characters.
  • ID: HTML attribute that provides a unique name for an element, used to manipulate it with JavaScript or CSS.
  • Class: HTML attribute that allows assigning a name to a group of elements, used to select them with JavaScript or CSS.
  • CSS: Cascading Style Sheets language used to define the appearance and layout of a web page.
  • JavaScript: Programming language used to add interaction and dynamism to web pages.
  • Browser: Software that allows users to view and interact with web pages.
  • Responsive Design: Practice of creating web pages that automatically adjust to the screen size and orientation of the device.

CSS

CSS (Cascading Style Sheets) - Lenguaje utilizado para definir la apariencia y el diseño de una página web.

  • Selector: CSS element that selects HTML elements to apply styles.
  • Property: Style assigned to an element, such as size or color.
  • Value: The value of a property, such as size in pixels or color in hexadecimal.
  • Rule: Combination of a selector, property, and value.
  • Declaration: Property and value within a CSS rule.
  • Cascade: The process by which rules are applied in order of specificity.
  • Priority: Determines which rules are applied in case of a conflict, prioritizing the more specific ones.
  • Box Model: Concept that describes how HTML elements are composed (margin, border, content).
  • Margin: Space between the element and adjacent content.
  • Padding: Space between the content of an element and its border.
  • Border: The boundary around an element's content.
  • Background: Property that defines the background color or image of an element.
  • Media Query: Function that applies styles based on the size and type of device.
  • Display: Property that controls how an element is displayed (block or inline).
  • Position: Property for positioning elements (static, fixed, or floating).
  • Float: Positions elements to the left or right of their container.
  • Clear: Controls the behavior of floating elements.
  • Z-index: Property that defines the depth of an element on the Z axis.
  • Hover: Pseudo-class that applies styles when the cursor is placed over an element.
  • Focus: Pseudo-class that applies styles when an element receives focus (e.g., a form field).
  • Transition: Defines the change of an element from one state to another.
  • Animation: Creates animations to move or change properties of an element.
  • Keyframes: Defines intermediate states of an animation.
  • Flexbox: Model for creating flexible layouts (adaptive rows and columns).
  • Grid: Model for creating two-dimensional layouts (adaptive rows and columns).
  • CSS Framework: Set of pre-defined rules to quickly create websites, such as Bootstrap.
  • CSS Preprocessor: Languages like Sass or Less to extend and customize CSS.
  • CSS Grid Layout: Model for creating flexible and adaptive two-dimensional layouts.

JavaScript

JavaScript - Lenguaje de programación de alto nivel utilizado para agregar interacción y dinamismo a las páginas web.

  • Variable: Memory space used to store values and assign names to those values.
  • Function: Reusable block of code that can be called with different parameters and returns a value.
  • Object: Set of properties and methods that represent an entity or abstract entity.
  • Array: Ordered and mutable collection of elements, similar to a list in other languages.
  • String: Sequence of characters enclosed in double or single quotes.
  • Number: Numeric data type that represents integer or decimal numbers.
  • Boolean: Data type that represents true or false values.
  • Conditional Statement: Control structure that executes code based on a boolean condition.
  • Loop: Control structure that repeats code execution while a condition is met.
  • If Statement: Conditional statement that executes a block of code if a condition is true.
  • Else Statement: Clause that executes if the condition in the "If" statement is not met.
  • Elif Statement: Clause that executes if the "If" and "Else" conditions are not met.
  • For Loop: Loop that iterates over a sequence of elements and executes code for each one.
  • While Loop: Loop that executes a block of code as long as a boolean condition is met.
  • Event: Occurrence in an object that triggers an action, such as a click or a change in a field.
  • Event Listener: Function that assigns an action to a specific event.
  • Document Ready: Event that occurs when the HTML document has fully loaded.
  • Window Object: Global object representing the browser window and its properties and methods.
  • Local Storage: Technology that allows storing local data in the browser to be retrieved later.
  • Session Storage: Stores session data in the browser as long as the session lasts.
  • Cookies: Small text files stored in the browser that allow tracking activity or storing data.
  • AJAX: Technology that allows sending HTTP requests without reloading the page, improving interaction with the server.
  • JSON: Lightweight data exchange format used for communication between client and server.
  • Callback Function: Function passed as an argument that is executed automatically when the task is completed.
  • Promise: Object representing an operation that may take time to complete, allowing success or failure to be handled.
  • Async/await: Pattern that simplifies the writing of asynchronous code using async and await functions.
  • Closure: Function that can access external variables even after those variables have left their scope.
