HTML

HTML - Lenguaje de marcado utilizado para crear páginas web y estructurar contenido en la web.

  • Tag: Elemento básico de HTML que define un elemento de la página web, como un título, texto, imagen o enlace.
  • Element: Componente básico de HTML que puede ser una etiqueta de apertura, una etiqueta de cierre o un contenido.
  • Attribute: Información adicional que se puede agregar a una etiqueta HTML para proporcionar información sobre el elemento.
  • Block Element: Elemento HTML que crea una nueva línea en la página y puede contener otros elementos.
  • Inline Element: Elemento HTML que se inserta dentro de otra etiqueta y no crea una nueva línea en la página.
  • Head: Sección de una página HTML que contiene información sobre el documento, como el título y las metas.
  • Body: Sección de una página HTML que contiene el contenido visible de la página.
  • Hyperlink: Elemento HTML que permite a los usuarios navegar entre páginas web.
  • Image: Elemento HTML que muestra una imagen en la página web.
  • List: Elemento HTML que permite mostrar una lista de elementos, como un menú de navegación o una lista de productos.
  • Form: Elemento HTML que permite recolectar información de entrada del usuario, como nombres, direcciones o contraseñas.
  • Input: Elemento HTML que permite al usuario ingresar información, como texto, números o seleccionar opciones.
  • Button: Elemento HTML que crea un botón en la página que puede ser utilizado para enviar formularios o ejecutar funciones.
  • Semantic HTML: Práctica de usar elementos HTML que proporcionan un significado claro, mejorando la accesibilidad y la indexación.
  • Header: Elemento HTML que contiene información de presentación y navegación en la parte superior de la página.
  • Footer: Elemento HTML que contiene información adicional en la parte inferior de la página.
  • Sidebar: Elemento HTML que contiene contenido secundario o relacionado en un área lateral de la página.
  • Section: Elemento HTML que define una sección o área de contenido en la página.
  • Article: Elemento HTML que representa un artículo o entrada independiente en una página.
  • Nav: Elemento HTML que crea un menú de navegación en la página.
  • Div: Elemento HTML genérico que permite agrupar y organizar otros elementos en la página.
  • Span: Elemento HTML genérico que permite agregar estilo o comportamiento a un conjunto de caracteres.
  • ID: Atributo HTML que proporciona un nombre único para un elemento, utilizado para manipularlo con JavaScript o CSS.
  • Class: Atributo HTML que permite asignar un nombre a un grupo de elementos, utilizado para seleccionarlos con JavaScript o CSS.
  • CSS: Lenguaje de hojas de estilo en cascada utilizado para definir la apariencia y el diseño de una página web.
  • JavaScript: Lenguaje de programación utilizado para agregar interacción y dinamismo a las páginas web.
  • Browser: Software que permite a los usuarios visualizar y interactuar con páginas web.
  • Responsive Design: Práctica de crear páginas web que se ajustan automáticamente al tamaño y la orientación de la pantalla del dispositivo.

CSS

CSS (Cascading Style Sheets) - Lenguaje utilizado para definir la apariencia y el diseño de una página web.

  • Selector: Elemento CSS que selecciona los elementos HTML para aplicar estilos.
  • Property: Estilo asignado a un elemento, como tamaño o color.
  • Value: Valor de una propiedad, como tamaño en píxeles o color en hexadecimal.
  • Rule: Combinación de un selector, propiedad y valor.
  • Declaration: Propiedad y valor dentro de una regla CSS.
  • Cascade: El proceso por el cual las reglas se aplican en orden de especificidad.
  • Priority: Establece qué reglas se aplican en caso de conflicto, priorizando las más específicas.
  • Box Model: Concepto que describe cómo se componen los elementos HTML (margen, borde, contenido).
  • Margin: Espacio entre el elemento y el contenido adyacente.
  • Padding: Espacio entre el contenido de un elemento y su borde.
  • Border: Límite alrededor del contenido de un elemento.
  • Background: Propiedad que define el color o imagen de fondo de un elemento.
  • Media Query: Función que aplica estilos según el tamaño y tipo de dispositivo.
  • Display: Propiedad que controla cómo se muestra un elemento (en bloque o en línea).
  • Position: Propiedad para posicionar elementos (estático, fijo, flotante).
  • Float: Posiciona elementos a la izquierda o derecha de su contenedor.
  • Clear: Controla el comportamiento de los elementos flotantes.
  • Z-index: Propiedad que define la profundidad de un elemento en el eje Z.
  • Hover: Pseudoclase que aplica estilos cuando el cursor se coloca sobre un elemento.
  • Focus: Pseudoclase que aplica estilos cuando un elemento recibe el foco (por ejemplo, un campo de formulario).
  • Transition: Define el cambio de un elemento de un estado a otro.
  • Animation: Crea animaciones para mover o cambiar propiedades de un elemento.
  • Keyframes: Define los estados intermedios de una animación.
  • Flexbox: Modelo para crear diseños flexibles (filas y columnas adaptativas).
  • Grid: Modelo para crear estructuras de dos dimensiones (filas y columnas) adaptativas.
  • CSS Framework: Conjunto de reglas predefinidas para crear sitios web rápidamente, como Bootstrap.
  • CSS Preprocessor: Lenguajes como Sass o Less para extender CSS y personalizarlo.
  • CSS Grid Layout: Modelo para crear layouts flexibles y adaptativos en dos dimensiones.

JavaScript

JavaScript - Lenguaje de programación de alto nivel utilizado para agregar interacción y dinamismo a las páginas web.

  • Variable: Espacio en memoria utilizado para almacenar valores y asignar nombres a esos valores.
  • Function: Bloque de código reutilizable que puede ser llamado con diferentes parámetros y devuelve un valor.
  • Object: Conjunto de propiedades y métodos que representan una entidad o entidad abstracta.
  • Array: Colección ordenada y mutable de elementos, similar a una lista en otros lenguajes.
  • String: Secuencia de caracteres entre comillas dobles o simples.
  • Number: Tipo de dato numérico que representa números enteros o decimales.
  • Boolean: Tipo de dato que representa valores verdaderos o falsos.
  • Conditional Statement: Estructura de control que ejecuta código según una condición booleana.
  • Loop: Estructura de control que repite la ejecución de código mientras se cumpla una condición.
  • If Statement: Instrucción condicional que ejecuta un bloque de código si se cumple una condición.
  • Else Statement: Cláusula que se ejecuta si la condición del "If" no se cumple.
  • Elif Statement: Cláusula que se ejecuta si la condición "If" y "Else" no se cumplen.
  • For Loop: Bucle que itera sobre una secuencia de elementos y ejecuta código para cada uno.
  • While Loop: Bucle que ejecuta un bloque de código mientras se cumpla una condición booleana.
  • Event: Ocurrencia en un objeto que desencadena una acción, como un click o un cambio en un campo.
  • Event Listener: Función que asigna una acción a un evento específico.
  • Document Ready: Evento que ocurre cuando el documento HTML ha sido completamente cargado.
  • Window Object: Objeto global que representa la ventana del navegador y sus propiedades y métodos.
  • Local Storage: Tecnología que permite almacenar datos locales en el navegador para ser recuperados después.
  • Session Storage: Almacena datos de sesión en el navegador mientras dure la sesión.
  • Cookies: Pequeños archivos de texto que se almacenan en el navegador y permiten rastrear actividad o almacenar datos.
  • AJAX: Tecnología que permite enviar solicitudes HTTP sin recargar la página, mejorando la interacción con el servidor.
  • JSON: Formato ligero de intercambio de datos utilizado para la comunicación entre cliente y servidor.
  • Callback Function: Función pasada como argumento que se ejecuta automáticamente cuando se completa la tarea.
  • Promise: Objeto que representa una operación que puede tardar en completarse y permite manejar éxito o error.
  • Async/await: Patrón que simplifica la escritura de código asincrónico usando funciones async y await.
  • Closure: Función que puede acceder a variables externas, incluso después de que esas variables hayan salido de su ámbito.
